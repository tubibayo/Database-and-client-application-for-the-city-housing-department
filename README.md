# Database-and-client-application-for-the-city-housing-department
Курсовая работа по базам данных. Разработана система для автоматизации учёта жилого фонда: управление районами, домами, помещениями, жильцами, платежами, услугами и формирование аналитических отчётов.
/n Курсовая.sql - запросы для создания спроектированной в рамках данной курсовой работы базы данных.
